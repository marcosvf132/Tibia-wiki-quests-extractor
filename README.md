# Tibia-wiki-quests-extractor
This tool can extract all tibia wiki registered quests and worldchanges into a lua table that contains the name of the quests with small informations like name, alias and version implementation (When possible)

# Why this exists
This tool was created by me for studies purpose only, not intented to have any meaning for anyone except the owner of this repository.

# Output
The output file is created on the executable path on a file named 'output.lua'

## Structure
```
|- Quest
   |- RookQuest
      | -
      | -
   |- WorldChange
      | -
      | -
   |- MainQuest
      | -
      | -
 ```
